## เทคโนโลยีที่ใช้

### Backend
- **AzuraCast** - ตัวจัดการ radio station แบบ open source ใช้งานง่ายมาก
- **Icecast** - Streaming server
- **Liquidsoap** - AutoDJ ที่เล่นเพลงอัตโนมัติ

### Frontend
- **Next.js 14** - เลือกใช้เพราะ deploy ง่าย performance ดี
- **React 18** - สำหรับสร้าง UI
- **Tailwind CSS** - เขียน CSS เร็วกว่าปกติเยอะ
- **TypeScript** - ช่วยลด bug ได้เยอะ

### Infrastructure
- **Docker** - ติดตั้งง่าย ไม่ต้องกังวลเรื่อง dependencies
- **Vercel** - deploy ฟรี auto deploy ทุกครั้งที่ push code

## แรงบันดาลใจ

ดีไซน์ได้แรงบันดาลใจจาก:
- Spotify - สำหรับ player controls
- Apple Music - สำหรับ layout
- SoundCloud - สำหรับ live indicator
